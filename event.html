11:00

  • сроки;
  • порядок подачи заявки;
  • пакет необходимых документов;
  • контакты для консультаций.
Алгоритм работы с микрозаймами, выданными до пандемии (отсрочка платежей):
  • порядок отсрочки выплат;
  • необходимые условия для отсрочки;
  • контакты для предоставления консультаций.
